    # GREYE Helm Chart

## Introduction

This repo contains the Helm chart for greye.

## Prerequisites

- Kubernetes 1.16+
- Helm 3.0+

## Installation

Add the GREYE repository:

```shell
helm repo add greye https://ftrigari.github.io/helm-charts
```
